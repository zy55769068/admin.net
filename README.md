# Admin.NET

#### 🎁 介绍
基于.NET6(Furion)/SqlSugar实现的通用管理平台，前端Vue3/Element-plus。整合最新技术，模块插件式开发，前后端分离，开箱即用。让.NET开发更简单，更通用，更流行。

#### 📖 使用说明
1.  支持各种数据库，后台配置文件自行修改（自动生成数据库及种子数据）
2.  前端运行步骤：1、安装依赖cnpm install 2、运行cnpm run dev 3、打包cnpm run build
