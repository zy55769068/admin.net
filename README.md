# Admin.NET

#### 🎁 介绍
基于.NET6/Furion/SqlSugar实现的通用管理平台，前端Vue3/Vben。整合最新技术，模块插件式开发，前后端分离，开箱即用。


#### 📖 使用说明

1.  支持各种数据库，后台配置文件自行修改（自动生成数据库及种子数据）
2.  前端运行步骤：1、yarn安装依赖 2、pnpm serve运行 3、pnpm build打包
